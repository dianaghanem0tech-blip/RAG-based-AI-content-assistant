# Requirements

## Python packages

- streamlit
- chromadb
- sentence-transformers
- pypdf
- python-docx
- ollama
- fpdf
- python-pptx
- requests

## System requirements

- Windows OS
- Python 3.10 or newer
- Enough disk space for `chroma_db` storage and generated outputs
- Internet access to download dependencies and fetch the approved Pearson PDF

## Optional

- Approved BTEC source files stored in a local `data` folder
- `ollama` model server running locally with the `llama3` model
